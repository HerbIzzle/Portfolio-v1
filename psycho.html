Psychotherapie Vöcklabruck - Komerzielle Webseite

Das Projekt entstand als Ich während meiner Ausbildung von einer Bekannten darauf angesprochen wurde ob Ich Ihre Webseite neu entwickelten möchte. Die vorherige Homepage war ein veraltetes Wordpress Template mit wenig guten Features. So bot sich mir die Gelegenheit mein erstes kommerzielles Projekt vom Start bis zum Deployment umzuseten.

Mein Ansatz war ein komplettes Redesign der Homepage mit einem professionellen Layout und verbesserten Inhalten. Weitere Features der Website beinhalten kleine Animationen on Scroll, ein Kontaktformular und Feature Boxen mit informativen Popups.

Die Seite ist Desktop first programmiert und für kleinere Ausgabemedien mittels Media Queries adaptiv angepasst.
