Kundenverwaltungs App - Angular

Eine einfache Applikation zur Verwaltung einer Kundendatenbank. Nutzer können KundInnen hinzufügen, bearbeiten oder entfernen. Dem Nutzer ist es auch möglich nach Vor- & Nachnahmen zu filtern.

Das Front End ist in Angular geschrieben.

Das Back End besteht aus einer Java Spring Boot App mit einer persistenten Apache Derby Datenbank.

Technologien & Konzepte:
  • - TypeScript
  • - Angular
  • - Java Spring Boot
  • - Apache Derby Datenbank
  • - Angular Material UI
  • - CRUD Operationen
  • - Amazon Web Service
